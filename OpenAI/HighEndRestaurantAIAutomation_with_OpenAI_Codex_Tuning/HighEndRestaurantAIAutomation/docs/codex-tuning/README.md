# Codex Tuning Documentation Index

**Purpose:** Index of all tuning markdown files for the project.

This folder contains operational instructions for Codex and human developers. Start with `AGENTS.md` at the repository root, then read the most relevant guide for your task.

## Recommended reading order

1. `AGENTS.md`
2. `skills.md`
3. `agents.md`
4. `prompts.md`
5. Task-specific tuning guide, such as `token_optimization.md`, `rag_optimization.md`, or `safety_guardrails.md`.

## Maintenance rule

When a new feature introduces a new agent, prompt, tool, memory type, or Azure service integration, update these docs in the same pull request.

# AI-102 Practice Mapping for the Project

**Purpose:** Maps repository features to Microsoft AI-102 study guide practice skills.

## Service selection practice

- Azure OpenAI: concierge response generation, embeddings, summarization.
- Azure AI Search: RAG retrieval over policies, menus, and event documents.
- Azure AI Content Safety: prompt and output moderation, blocklists, prompt injection checks.
- Azure AI Language: PII, sentiment, NER, CLU-style intent classification.
- Azure AI Translator: multilingual guest interaction.
- Azure AI Speech: voice concierge STT/TTS.
- Azure AI Vision: menu photo or dining room image analysis.
- Document Intelligence: supplier invoices, event forms, menu PDFs.
